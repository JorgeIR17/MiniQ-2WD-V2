void HAL_bateria_mostrar_nivel ( )

Muestra el valor del porcentaje de batería.

Esta función hace uso del led RGB para mostrar mediante colores la batería restante. Si se encuentra sobre el 40%, mostrará el color verde. Si se encuentra entre el 40% y el 15%, mostrará el color amarillo. Si el nivel se encuentra por debajo del 15%, el led parpadeará en color rojo.
